📖 Todo sobre la etiqueta HEAD

🧠 ¿Qué es el HEAD?

El <head> es como el "cerebro invisible" de tu página web. Contiene información importante que el navegador necesita, pero que los visitantes NO ven directamente en la página.

📍 Ubicación: El HEAD siempre va entre <html> y <body>

<html>
  <head>
    ← Aquí va toda la metadata
  </head>
  <body>
    ← Aquí va el contenido visible
  </body>
</html>

💡 Analogías para entender el HEAD

1. Como la etiqueta de un producto:

  • 📦 HEAD = Etiqueta con ingredientes, precio, marca
  • 👀 BODY = El producto que ves y usas

2. Como el carnet de identidad:

  • 🆔 HEAD = Datos del documento (nombre, fecha, tipo)
  • 🖼️ BODY = Tu foto y contenido visible

3. Como una carta:

  • ✉️ HEAD = El sobre con dirección y remitente
  • 📄 BODY = El mensaje dentro de la carta

⚡ Dato Importante

Aunque los usuarios NO ven el HEAD directamente, es SÚPER importante porque:

  • 🔍 Google lee el HEAD para indexar tu sitio
  • 📱 Navegadores lo usan para mostrar tu página correctamente
  • 🌐 Redes sociales lo leen cuando compartes un link
  • ♿ Lectores de pantalla lo usan para personas con discapacidad visual

📦 Etiquetas que van en el HEAD:

<title>

El título que aparece en la pestaña del navegador. ¡Es lo primero que ven los usuarios!

<meta charset>

Define el conjunto de caracteres. Con "UTF-8" puedes usar acentos: á, é, ñ, ¿, ¡

<meta viewport>

Hace que la página se vea bien en celulares. ¡Muy importante hoy en día!

<meta description>

Una descripción corta de tu página. Google la muestra en los resultados de búsqueda.

<link>

Conecta archivos externos como hojas de estilo CSS o el favicon (icono de la pestaña).

<style>

Permite agregar estilos CSS directamente en el HTML. ¡Colores, tamaños y más!

🔤 Entendiendo charset="UTF-8"

UTF-8 es un sistema de codificación que permite usar caracteres especiales de todos los idiomas:

  • 🇪🇸 Español: á, é, í, ó, ú, ñ, ¿, ¡
  • 🇫🇷 Francés: ç, é, è, ê
  • 🇯🇵 Japonés: あ, い, う
  • 😀 Emojis: 🎉, ❤️, 🚀

🌍 Ejemplos del Mundo Real

🔍 HEAD en sitios famosos:

📺 YouTube

<meta charset="utf-8">
<title>YouTube</title>
<meta name="description" content="Disfruta videos...">
<meta name="viewport" content="width=device-width">

✅ Charset, title, description y viewport

🛍️ Amazon

<meta charset="utf-8">
<title>Amazon.com: Compras en línea</title>
<meta name="keywords" content="Amazon, compras">
<link rel="stylesheet" href="estilos.css">

✅ Incluye keywords y link a CSS externo

🎯 Lecciones clave:

  • TODOS usan charset UTF-8
  • El title es crucial para SEO
  • La description aparece en Google
  • El viewport es obligatorio para móviles
🚀 Meta Tags Avanzadas (Bonus)

🎨 Open Graph (Redes sociales)

<meta property="og:title" content="Mi Artículo Increíble">
<meta property="og:description" content="Descripción al compartir">
<meta property="og:image" content="preview.jpg">

📱 Controla cómo se ve tu link cuando lo compartes en Facebook, WhatsApp o Twitter

🤖 Meta Robots (Para Google)

<meta name="robots" content="index, follow">

🔍 Le dice a Google si puede indexar (guardar) tu página y seguir los links
• index = Permitir en búsquedas
• noindex = NO mostrar en Google
• follow = Seguir los links

🎭 Theme Color (Móviles)

<meta name="theme-color" content="#3498db">

📱 Cambia el color de la barra superior del navegador en móviles para que combine con tu sitio

📱 Apple Touch Icon

<link rel="apple-touch-icon" href="icon-180x180.png">

🍎 Icono cuando alguien agrega tu sitio a la pantalla de inicio de iPhone/iPad
