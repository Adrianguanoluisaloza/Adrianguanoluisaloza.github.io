📖 Introducción a JavaScript

⚡ ¿Qué es JavaScript?

JavaScript es el lenguaje de programación que hace que las páginas web sean interactivas. Con JS puedes: responder a clics, cambiar contenido, validar formularios, crear juegos y mucho más.

💡 HTML + CSS + JS

HTML = Estructura (esqueleto)

CSS = Estilo (ropa y maquillaje)

JavaScript = Comportamiento (movimiento y vida)

🧱 Conceptos básicos:

📦
Variables
let, const
🔢
Tipos de datos
string, number
⚙️
Funciones
function()
🎯
Eventos
onclick, etc.

📦 Variables (guardar datos):

// Crear variables
let nombre = "María";
let edad = 12;
const PI = 3.14159;

// Mostrar en consola
console.log(nombre);
console.log("Tengo " + edad + " años");

⚙️ Funciones (acciones):

// Crear una función
function saludar(nombre) {
    alert("¡Hola, " + nombre + "!");
}

// Llamar la función
saludar("Carlos");

🎯 Seleccionar elementos:

// Obtener elemento por ID
let titulo = document.getElementById("miTitulo");

// Cambiar contenido
titulo.textContent = "¡Nuevo título!";

// Cambiar estilo
titulo.style.color = "red";
