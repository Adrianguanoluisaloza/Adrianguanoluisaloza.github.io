💾 ¿Qué es LocalStorage?

LocalStorage es como una memoria especial del navegador donde puedes guardar información. ¡Lo mejor es que la información se queda guardada incluso si cierras el navegador!

💡 Piensa en LocalStorage como...

¡Un pequeño armario en tu navegador donde puedes guardar notas, puntuaciones, preferencias y más!

🛠️ Métodos principales:

localStorage.setItem("clave", "valor")
Guarda un dato
localStorage.getItem("clave")
Lee un dato
localStorage.removeItem("clave")
Borra un dato
localStorage.clear()
Borra TODO

📝 Ejemplo práctico:

// Guardar
localStorage.setItem("nombre", "Ana");
localStorage.setItem("edad", "9");

// Leer
let nombre = localStorage.getItem("nombre");
console.log(nombre); // "Ana"

// Borrar uno
localStorage.removeItem("edad");

// Borrar todo
localStorage.clear();

⚠️ Importante:

LocalStorage solo guarda texto (strings). Si quieres guardar números u objetos, usa JSON.stringify() para guardar y JSON.parse() para leer.
