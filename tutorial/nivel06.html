📖 Aprende sobre Formularios

📝 ¿Qué es un Formulario?

Los formularios permiten a los usuarios ingresar datos. Se usan para login, contacto, búsquedas, encuestas y mucho más.

💡 Estructura básica

Un formulario usa <form> como contenedor, <input> para campos, y <label> para etiquetas descriptivas.

📋 Atributos importantes:

  • name - Nombre del campo (para enviar datos)
  • placeholder - Texto de ayuda dentro del campo
  • required - Campo obligatorio
  • value - Valor predeterminado
  • disabled - Campo deshabilitado

📨 Otros elementos de formulario:

  • <textarea> - Área de texto grande
  • <select> - Lista desplegable
  • <button> - Botón de envío
